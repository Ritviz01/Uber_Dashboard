# 🚖 Uber Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Uber Analytics Dashboard** built using Power BI. The dashboard provides meaningful insights into Uber ride data, helping analyze booking trends, revenue patterns, and ride behavior.

---

## 🎯 Objectives

* Analyze total rides and revenue
* Identify peak booking hours
* Study ride distance patterns
* Understand customer demand trends

---

## 📊 Key Features

* 📈 Total Bookings & Revenue KPIs
* 🕒 Hour-wise Ride Analysis
* 📍 Location-based Insights
* 🚗 Ride Distance Distribution
* 📅 Date-wise Trends Visualization

---

## 🛠 Tools & Technologies Used

* **Power BI** – Dashboard creation & visualization
* **DAX (Data Analysis Expressions)** – Calculations & measures
* **Excel / CSV Dataset** – Data source
* **Data Cleaning & Transformation**

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Interact with filters and visuals to explore insights

---

## 💡 Business Insights

* Peak ride demand observed during evening hours
* Higher revenue generated from long-distance trips
* Significant variation in ride frequency across time slots
